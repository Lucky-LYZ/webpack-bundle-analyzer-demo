# webpack-bundle-analyzer-demo

探索 webpack 的可视化资源分析工具 webpack-bundle-analyzer 的使用

## 命令

```
- `npm run start` 命令，用于启动项目；
- `npm run build` 命令，用于打包项目；
- `npm run analyz` 命令，用于启动webpack-bundle-analyzer分析打包结果中各个依赖的占用情况；
```
